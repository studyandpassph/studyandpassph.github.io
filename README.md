# Study and Pass PH

Static site template for GitHub Pages (HTML/CSS/JS).

## Quick start
1. Upload all files to your GitHub repo.
2. GitHub repo Settings → Pages → Deploy from branch → main / root.
3. Visit your site URL.

## Monetization placeholders
- Ad slots: replace the placeholder boxes with your AdSense code after approval.
- Affiliate links: add to reviewer and blog pages with disclosures.
